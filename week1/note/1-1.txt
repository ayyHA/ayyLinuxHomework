ls指令 list directory contents: 列出文件夹下的文件
  注意：第一列为d表示为文件夹
        第一列为-表示为文件
        第一列为l表示为链接
        最末一列为x表示为可执行文件

        -l 表示列出文件的详细信息
     
mkdir指令: 建立空目录

rmdir指令: 删除空目录 与 rm -d 相同

cd ~: 跳转到当前用户目录下

man指令: 查询指令内容，与help相似，但更为详细

vi/vim指令: 创建文件

i/o/a: 进入编辑模式，通过esc返回初始模式
::进入命令行模式，通过esc返回初始模式，
:wq 写入并退出
:q 退出不保存，但需无更改文件内容
:q! 退出不保存

gcc -E 文件名.c > 文件名.i : 预编译
gcc -S 文件名.i -o 文件名.s : 产生汇编文件
gcc 文件名.c -o 文件名 : 产生可执行文件

文件名 2>文件名.dat : 将错误信息保存
注：0-255共256位，其中0-stdi 1-stdo 2-stderr 别的并不固定
    linux一切皆文件，不以文件后缀名区分文件

文本文件(ASCII) 查看指令有如下三条：cat -n : 输入内容及附上行号 
				    more
				    less
文本文件(二进制) 查看指令有如下两条： readelf
				      objdump

pwd指令: 查看当前目录

rm指令:
	-r 删除非空目录
	-f 不提示，强制删除
	-i 提示删除信息

cp指令:
	cp 源文件 目标目录
	注:若没有对文件重命名，则为同名拷贝，反之为异名拷贝
	   支持类似模糊匹配似的拷贝，即位批量拷贝，但只允许拷到文件夹下，不允许改名

	co -r/-R ：拷贝目录
